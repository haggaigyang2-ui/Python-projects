# Engineering Tech Blog

A modern, responsive tech blog website built with HTML and CSS, dedicated to sharing knowledge about software engineering, DevOps, and cloud technologies.

## Features

✨ **Responsive Design** - Fully responsive on desktop, tablet, and mobile devices  
📝 **Blog Section** - Display articles with categories and dates  
🎨 **Modern UI** - Clean, professional design with gradient backgrounds  
📧 **Contact Form** - Get in touch with visitors  
⚡ **Fast Loading** - Pure HTML/CSS, no dependencies  
🔗 **Navigation** - Smooth scrolling to different sections  

## How to Use

1. Open `index.html` in your browser
2. Customize the content as needed
3. Deploy to GitHub Pages, Netlify, or Vercel

## Customization

- **Colors**: Edit in `styles.css` - Primary color: `#667eea`, Secondary: `#764ba2`
- **Content**: Modify text directly in `index.html`
- **Blog Posts**: Duplicate a `.blog-card` element and edit

## Deployment

### GitHub Pages
1. Go to Settings → Pages
2. Select `main` branch
3. Your site will be live at `https://haggaigyang2-ui.github.io/Python-projects`

### Netlify / Vercel
Connect your GitHub repo for automatic deployment.

## File Sizes

- `index.html`: ~15KB
- `styles.css`: ~10KB
- **Total**: ~25KB (Super Fast!)
